Study moves that make this stick

1. Say the timeline first.
Before conjugating, say: now, finished, ongoing, or future.
2. Build one verb four ways.
hablo → hablé → hablaba → hablaré. This teaches the tense pattern faster than isolated charts.
3. Anchor examples in places you know.
Use San Juan, la playa, Viejo San Juan, clase, and casa so the sentence feels real.
4. For preterite vs imperfect, draw it.
Dot for completed. Line for repeated, ongoing, or background.
5. For fui/fue/fueron, look for movement.
If someone went somewhere, it is ir. If it describes what something was, it is ser.
